# YouTube URL Scraper Chrome Extension

A simple Chrome extension to scrape video URLs from a YouTube playlist or your liked videos. This tool allows you to easily extract and download the URLs of selected videos from YouTube.

## Features

* Extract YouTube video URLs from playlists or liked videos.
* Download selected video URLs with just a few clicks.
* Lightweight and easy to use.

## Installation

### 1. Clone the repository:

```bash
git clone https://github.com/AAlkiyumi/Youtube-URL-Scraper-Chrome-Extension.git
```

### 2. Install the extension in Chrome:

1. Go to `chrome://extensions` in your browser.
2. Enable "Developer mode" (toggle in the top right corner).
3. Click on the "Load unpacked" button.
4. Select the folder where you cloned the repository.

### 3. Extract YouTube URLs:

1. Open your YouTube liked videos or playlist.
2. Click on the extension icon in the toolbar.
3. Select the videos you want to extract URLs for.
4. Click the "Download Selected URLs" button to save the links.

## Usage

* Once installed, you can use the extension anytime to grab URLs from YouTube playlists or liked videos.
* Simply open the playlist or video page, click the extension icon, select the videos, and download the URLs.
